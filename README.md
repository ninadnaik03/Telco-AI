# Telco Troubleshooting Assistant

## Overview
This project explores the use of large language models (LLMs) to assist with diagnosing telecom network issues described in natural language. The goal is to study how LLMs can reason about domain-specific problems and produce structured diagnostic outputs.

## Current Status
- Prototype LLM-based troubleshooting pipeline
- Structured diagnosis output (JSON)
- Retrieval-based context grounding (early stage)

## Design Philosophy
- Research-first, not production-ready
- Emphasis on robustness and interpretability
- Explicit handling of LLM limitations

## Planned Extensions
- Improved retrieval grounding
- Evaluation against baseline models
- Iterative (agent-like) troubleshooting
- Domain-specific fine-tuning
